# Rainbow Gantt V2 - 使用說明

## 五大功能頁面

| 圖標 | 名稱 | 功能 |
|------|------|------|
| 📊 | ACTIVE_PROJECTS | 進行中的專案（甘特圖/列表視圖） |
| ✅ | COMPLETED_LOG | 已完成專案記錄 |
| 🧪 | POC_PROTOCOLS | 概念驗證提案 |
| 📅 | CALENDAR | 雙人日曆視圖 |
| 📖 | KNOWLEDGE_BASE | 學習筆記與知識庫 |

---

## ACTIVE_PROJECTS（進行中專案）

### 視圖切換
- **GANTT**：甘特圖視圖
- **LIST**：列表視圖

### 人員篩選
- **ALL**：顯示所有任務
- **JANUS**：只顯示 Janus 的任務
- **JOSEPH**：只顯示 Joseph 的任務

### 甘特圖操作
- `< PREV`：上個月
- `NEXT >`：下個月
- `TODAY`：跳轉到當前月份
- 點擊任務條可編輯

### 優先級標記
- 🔴 URGENT
- 🟠 HIGH
- 🟡 MEDIUM
- ⚪ LOW

---

## COMPLETED_LOG（已完成專案）

### 顯示欄位
- PROTOCOL_NAME：專案名稱
- AGENT：負責人
- CATEGORY：類別標籤
- START_DATE：開始日期
- END_DATE：完成日期
- DURATION：執行天數（自動計算）
- ACTIONS：刪除按鈕

### 操作
- 點擊 `+ ADD_ENTRY` 添加已完成項目
- 點擊垃圾桶圖標刪除記錄

---

## POC_PROTOCOLS（概念驗證）

### 卡片顯示
- 標題
- 狀態標籤（PENDING / IN_PROGRESS）
- 類別標籤
- 負責人
- 利益相關者（顯示前 3 個）
- 描述

### 操作
- 點擊 `+ NEW_PROTOCOL` 新增
- 點擊卡片編輯
- 點擊垃圾桶刪除

---

## CALENDAR（雙人日曆）

### 視角切換
- **JA**（Janus）
- **JO**（Joseph）

### 月份導航
- `<`：上個月
- `>`：下個月

### 添加/編輯內容
1. 點擊日期格子進入編輯模式
2. 輸入內容或上傳圖片
3. 點擊 **SAVE** 保存

### 圖片操作
- **上傳**：點擊 UPLOAD IMAGE 按鈕
- **查看**：點擊縮圖全屏查看
- **刪除**：編輯模式下點擊圖片右上角 ✗

---

## KNOWLEDGE_BASE（知識庫）

### 筆記卡片
- 標題
- 內容預覽（Markdown 渲染）
- 相關專案標籤
- 最後修改時間

### 操作
- 點擊 `+ NEW_ENTRY` 新增筆記
- 點擊 📄 圖標編輯
- 點擊 🗑️ 圖標刪除

### 編輯模式
- **標題**：必填
- **內容**：支持 Markdown 格式
- **附件**：Link / File / Image
- **關聯專案**：點擊 `+ Add Related Protocol` 選擇相關專案

---

## 任務管理

### 新增任務
- 從 Active Projects：點擊 `+ NEW_TASK`
- 從 POC/Completed：點擊對應頁面的新增按鈕

### 任務欄位

**基本資訊**：
- 任務名稱（必填）
- 負責人（必填）
- 狀態：Pending / In Progress / Completed / Blocked（必填）
- 優先級：Urgent / High / Medium / Low
- 類別（必填）

**時間設定**：
- 開始日期（必填，格式：YYYY-MM-DD）
- 結束日期（必填）
- 進度（0-100%）

**POC 專屬**：
- 是否為 POC（勾選後顯示在 POC_PROTOCOLS）
- 利益相關者（姓名 + 角色）
- 描述

**附件**：
- 類型：Link / File / Image
- 圖片可點擊放大查看

### 編輯任務
- 從甘特圖：點擊任務條
- 從列表視圖：點擊任務行
- 從 POC 頁面：點擊卡片

### 刪除任務
1. 打開任務編輯面板
2. 滾動到底部點擊 **DELETE**
3. 刪除操作無法復原

---

## 人員管理

### 打開設定
點擊右上角 **⚙️ Settings**

### 操作
- **新增人員**：輸入名字後點擊 `+ ADD`
- **編輯名稱**：點擊 ✏️ 編輯，Enter 保存，Esc 取消
  - 自動同步所有關聯任務的 owner 欄位
- **刪除人員**：點擊 🗑️ 刪除
  - ⚠️ 關聯任務的 owner 會變成空白

### 關閉設定
- 點擊右上角 ✗
- 點擊面板外的遮罩區域
- 按 Esc 鍵

## 疑難排解

### 任務不顯示在甘特圖上？
1. 檢查任務是否勾選「是否為 POC」（POC 任務不顯示在甘特圖）
2. 檢查任務日期是否在當前月份範圍內
3. 檢查是否設置了開始和結束日期

### 人員篩選後看不到任務？
- 確認任務的 owner 欄位是否設置
- 確認 owner 名稱拼寫是否正確（區分大小寫）

### 日曆內容保存後消失？
- 確認點擊了 SAVE 而非 CANCEL
- 檢查切換的人員是否正確（JA/JO）
- 刷新頁面重新載入

### 圖片上傳失敗？
- 確認圖片大小不超過 5MB
- 確認圖片格式為常見圖片格式
- 檢查網路連線

### 甘特圖任務條顯示不完整？
任務跨多個月時，只顯示當前月份的部分，使用 PREV/NEXT 切換月份查看完整任務。
